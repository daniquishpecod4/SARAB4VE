# SARA
## Sistema de Apoyo y Respuesta Accesible

SARA es una plataforma desarrollada para responder a emergencias y desastres naturales mediante la coordinación rápida de ayuda entre ciudadanos, voluntarios, organizaciones y organismos de respuesta.

Nace durante un hackatón como respuesta a un terremoto ocurrido hace 48 horas, donde miles de personas necesitan asistencia inmediata y la información crítica debe fluir de forma rápida, organizada e inclusiva.

## Problema

Después de un desastre:

- La información está dispersa.
- Las personas vulnerables no son identificadas rápidamente.
- La ayuda tarda en llegar.
- No existe una coordinación eficiente entre ciudadanos y organizaciones.

## Solución

SARA permite:

- Solicitar ayuda geolocalizada.
- Encontrar refugios y centros de apoyo.
- Conectar voluntarios con personas afectadas.
- Coordinar recursos críticos.
- Priorizar emergencias.
- Centralizar la información para organismos de respuesta.

## Usuarios

- Ciudadanos
- Personas con discapacidad
- Adultos mayores
- Voluntarios
- ONGs
- Protección Civil
- Bomberos
- Cruz Roja
- Autoridades

## MVP

- Registro de usuarios
- Botón SOS
- Solicitud de ayuda
- Mapa de incidentes
- Panel de voluntarios
- Seguimiento de solicitudes

## Tecnologías

- Flutter
- Firebase
- Google Maps / OpenStreetMap
- Supabase
- FastAPI
- PostgreSQL

## Objetivo

Reducir el tiempo entre una solicitud de ayuda y la llegada del recurso adecuado.
