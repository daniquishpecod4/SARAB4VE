# Registro de Cambios (Changelog)

Todos los cambios notables de SARA se documentan aquí.

El formato se basa en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/)
y el proyecto sigue [Versionado Semántico](https://semver.org/lang/es/).

## [Sin publicar]

### Agregado
- Estructura inicial de configuración del repositorio.
- Guía de contribución (`CONTRIBUTING.md`).
- Código de conducta (`CODE_OF_CONDUCT.md`).
- Política de seguridad (`SECURITY.md`).
- Plantillas para issues y pull requests.
- Licencia MIT.

---

> **Cómo usar este archivo:** cada vez que se hace un cambio importante, se agrega
> una línea bajo "Sin publicar". Cuando se lanza una versión, esas líneas se mueven
> a una nueva sección con el número de versión y la fecha. Ejemplo:
>
> ## [0.1.0] - 2026-07-01
> ### Agregado
> - Botón SOS funcional.
